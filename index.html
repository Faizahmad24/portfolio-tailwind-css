Portfolio

Latest Projects

I have had the contributed to various projects, both profesionally and personally. The following projects I've been involved in :

LuxSpace

LuxSpace is an e-commerce that sells kind of property and furniture. LuxSpace is developed with React JS library and Tailwind CSS for the Front End. In the meantime LuxSpace has Back End data from mocking server API.

Shopping Notation

Shopping Notation is a mini project based on website for making shopping notation such as deleting note, adding note according to the quantity from the user and filtering data notation by order input, name, and check. Shopping notation that made using JavaScript, React JS and CSS 3.

Shopping Cart

Shopping Cart is a mini project based on website. Shopping Cart has feature to display the shopping lists user. Shopping Cart also can add new shopping list and can reduce and add the quantity of each user's shopping items. Shopping Cart that made using JavaScript, React JS and CSS 3.

News Portal

News Portal is a mini project based on website. News Portal made for showing news to user according category that user want. News Portal has backend datas from consume public API newsapi.org. News Portal that made using JavaScript, React JS and CSS 3.

Integrated News Room

Integrated Newsroom (INR) is a system based on website like Content Management System (CMS) for news production that made using CodeIgniter 3.

Relawan Filantropi Indonesia (RFI)

RFI is a website that facilitates volunteers to market humanitarian campaign programs, qurban campaigns, zakat campaigns, where the campaign programs come from ACT's websites called Indonesia Dermawan, Gobal Qurban, and Global Zakat. RFI was created using the PHP Framework Laravel, Javascript, JQuery, and MySQL as the database.

Pasar Sedekah

Pasar Sedekah is a Market Place that connecting between buyer and seller like any other Market Place. Pasar Sedekah was created using the PHP Framework Laravel, Javascript, JQuery, and MySQL as the database.
